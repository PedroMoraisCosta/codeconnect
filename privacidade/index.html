2. Que dados recolhemos

2.1 Dados da pessoa identificada

  • Nome completo
  • Fotografia (opcional)
  • Condição de saúde ou necessidade especial
  • Instruções em caso de emergência

2.2 Dados dos responsáveis

  • Nome completo
  • Número de telefone

2.3 Dados de utilização

Não recolhemos dados desnecessários. Apenas o essencial para resposta em emergência.

3. Finalidade do tratamento

  • Identificação em emergência
  • Contacto com responsáveis
  • Gestão do serviço

Os dados não são usados para fins comerciais.

4. Base legal

Tratamento baseado em:

  • Consentimento explícito
  • Interesses vitais em emergência

Para menores, o consentimento é dado pelos responsáveis legais.

5. Dados sensíveis

  • Acesso restrito via QR Code ou NFC
  • Não partilhados sem consentimento
  • Possibilidade de remoção a qualquer momento

6. Segurança

  • Dados armazenados na UE
  • Proteção contra acesso não autorizado
  • Encriptação dos dados

7. Conservação

Os dados são mantidos enquanto o serviço estiver ativo.

  • Eliminação até 30 dias após cancelamento
  • Sem retenção após esse prazo

8. Direitos do titular

  • Acesso
  • Retificação
  • Eliminação
  • Limitação
  • Portabilidade
  • Oposição
